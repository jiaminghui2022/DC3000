# DC3000 侵染数据分析系统

专为拟南芥 *Pst* DC3000 侵染实验设计的一站式数据分析工具。从原始菌落数录入到出版级统计图表，全流程自动化。

**零安装、零依赖** — 浏览器直接打开 HTML 文件即可使用。

## 功能特性

- **数据录入** — 支持多基因型、多重复的菌落数据录入，自动选择最佳稀释梯度（15-100 菌落范围）
- **CFU 自动计算** — 基于打孔器直径、圆片数、点板体积等参数自动计算 CFU/cm² 并取 log₁₀
- **异常值剔除** — 支持 IQR 法和 Z-Score 法，被剔除的数据点在图表中以半透明红色标出
- **统计检验** — One-way ANOVA F 检验 → Welch's t-test 两两比较 → Bonferroni 校正 → CLD 显著性字母
- **三种图表** — 柱状图 + 散点、箱线图 + 散点、小提琴图 + 散点（Catmull-Rom 样条平滑曲线）
- **六种配色** — 暗黑风格 × 3 + 学术白底 × 3，适配不同投稿需求
- **数据导出** — 增强版 CSV（含实验参数元信息、原始数据、统计摘要、p-value 矩阵）、SVG 矢量图、PNG 300dpi
- **数据导入** — 支持导入之前导出的 CSV 文件，断点续做
- **自动保存** — 输入数据自动保存至浏览器本地存储，防止误关丢失
- **响应式设计** — 支持桌面端和移动端

## 使用方法

1. 下载 `dc3000_analysis_v2.html`，用浏览器打开
2. 设置实验参数（打孔器直径、圆片数、点板体积等）
3. 输入基因型名称和重复数，点击"添加组别"
4. 在表格中填入各稀释梯度的菌落数
5. 点击"开始计算与统计分析"
6. 在图表旁切换图表类型和配色方案
7. 导出 CSV 报告和图表文件

## 计算公式

```
CFU/cm² = (菌落数 × 稀释倍数 × (1000 / 点板体积µl)) / (圆片数 × π × (半径cm)²)
```

## 统计方法

| 步骤 | 方法 | 说明 |
|------|------|------|
| 整体检验 | One-way ANOVA | F 检验判断组间是否存在显著差异 |
| 两两比较 | Welch's t-test | 不假设等方差，适合样本量不等的组间比较 |
| 多重校正 | Bonferroni | p × C(k,2)，控制家族错误率 |
| 字母分配 | Compact Letter Display | 基于聚类算法的显著性字母标注 |

## 技术栈

- **UI**: 原生 HTML/CSS（Glassmorphism 暗黑风格）
- **图表**: 原生 SVG 手绘（无第三方图表库）
- **统计**: [jStat.js](https://github.com/jstat/jstat)（ANOVA、t-test、KDE）
- **图标**: [Lucide Icons](https://lucide.dev/)
- **字体**: [Outfit](https://fonts.google.com/specimen/Outfit)

## 示例

### 数据录入界面
录入各基因型在不同稀释梯度下的菌落数，系统自动高亮最佳梯度（绿色边框）。

### 统计图表
支持柱状图、箱线图、小提琴图三种类型，可实时切换配色方案。

### CSV 报告
导出的 CSV 包含完整的实验参数元信息、原始数据、统计摘要和 p-value 两两比较矩阵。
